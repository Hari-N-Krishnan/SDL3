MY SDL3 Projects following the SDL3 series on youtube by Mike Shah. Note that you have to have the SDL3.dll in your executable folder and
compile a suitable exe with something similar from build.txt. Example: g++ myProgram.cpp -o output.exe <PASTE IN BUILD.TXT'S LINE>.
Now just double click on the exe from your file explorer and it should run.

SDL3 OFFICIAL REPO https://github.com/libsdl-org/sdlwiki
TOOLCHAIN FOR COMPILATION https://www.mingw-w64.org/downloads/#w64devkit
